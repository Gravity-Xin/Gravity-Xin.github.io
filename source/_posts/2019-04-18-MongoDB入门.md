BSON结构的、面向文档的NoSQL

## 基本概念

- DataBase: 数据库，包含很多Collection

- Collection: 数据集合，包含很多Document

- Document: BSON格式的文档，是可以操作的最小单位

## 安装

- 启动参数
  - `--config`或`-f`: 配置文件路径
  - `--logpath`: 日志文件路径
  - `--dbpath`: 数据存储路径
  - `--auth`: 启动用户认证
  - `--port`: 监听的端口号

- Docker
  - `docker container run -d -p 27017:27017 -v mongo_data:/data/db --name MyMongo mongo`
